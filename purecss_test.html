The advantages/disadvatages of cloud-based development environments

There are many services out there that enable you to develop and test code in the cloud. I find that both local and cloud-based environments have their pros and cons. One problem I've ran into is importing a github repository into Codesandbox.io

I tried using the appending the github URL with "box" trick. I appended "box" in between "github" and ".com" in the URL, clicked enter, and lo and behold, the template loaded but absolutely failed to load correctly in the preview pane.

Emmet abbreviations work here in the online editor, which is nice, but a lot of other things are missing. I tend to prefer the easy UI available in VSC and Atom, and the ability to add snippets to make it easy to type out code blocks that you use often.

If there's a way to do that here in the cloud, that would make me more prone to use this environment.
